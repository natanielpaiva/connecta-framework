# Connecta Framework

The Framework for Connecta Java applications.

## Install

...
```
$ mvn install:install-file -Dfile=connecta-connectors/lib/ojdbc7.jar -DgroupId=com.oracle -DartifactId=ojdbc7 -Dversion=12.1.0.2 -Dpackaging=jar
```

```
$ mvn clean install
```
